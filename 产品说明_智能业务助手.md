# 产品说明｜智能业务助手

## 1. 目标用户
- 小微企业主、区域门店经营者、跨境电商卖家
- 已有稳定经营数据，但缺少专业融资顾问
- 对“缺钱扩张”场景敏感，希望快速得到是否可做、做多大、风险在哪

## 2. 核心场景
企业客户打开 AI Agent 控制台，输入：“我下个月想扩大业务规模，但资金不够，怎么办？”

系统需要在 3 轮对话内完成 5 件事：
- 识别融资/资金需求意图
- 用自然对话补齐必要经营信息
- 给出具体方案推荐，而不是泛泛建议
- 透明展示风险项与阻断项
- 把用户引导到正式申请前的下一步动作

## 3. 核心价值
- 对用户：更快知道“我现在能不能做、适合做多少”
- 对业务：把高意向用户从咨询转成申请线索
- 对风控：把风险暴露前置到对话内，而不是提交后再拒绝
- 对合规：保留审计记录，避免 AI 越权代替审批

## 4. 转化漏斗
1. 用户输入融资相关表达
2. 系统命中融资意图
3. 用户补齐月营业额、库存周转、目标金额
4. 系统输出推荐方案或风险阻断
5. 用户点击“准备资料 / 提交申请”

关键转化指标：
- 意图识别命中率
- 补齐信息完成率
- 推荐方案阅读率
- CTA 点击率
- 高风险用户流失率

## 5. 上线计划
### Phase 1：Demo 验证
- 用单页 HTML 演示标准流程和异常分支
- 让销售、产品、风控共同确认对话话术和边界

### Phase 2：灰度上线
- 接入真实用户对话日志
- 只做建议，不开放自动申请
- 验证最少必要字段和推荐命中率

### Phase 3：正式接申请
- 增加“资料清单自动带入”
- 对接申请系统，但审批仍由正式流程完成
- 增强审计、埋点和运营补救策略
